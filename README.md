**Exo**planet **T**ransited **Ho**st **M**odelling **F**ramework
___


**exoTHOMF** is an easy to use and flexible package for fitting a variety of quantities related to exoplanet host stars,
for example: stellar spectra, star spot contrast spectra, or stellar contamination.

## Documentation
An installation guide, code documentation, and tutorials are included in the [exoTHOMF documentation]().
